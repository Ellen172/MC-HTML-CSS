Estutrura HTML

<!Doctype html> → Define a versão do código utilizado

<html> → Tag de abertura para o documento html

<head> → Abertura do cabeçalho, onde se encontram as especificação do código

<title> Nome da página </title> → Indica o titulo do site/navegador (que aparece na guia)

<meta charset="utf-8"> → Indica o campo de caracteres

</head> → Tag de fechamento do cabeçalho

<body> → Abertura do corpo do site, onde serão dispostos todos os principais elementos

</body> → Tag de fechamento do corpo do site

<footer> → Abertura do rodapé do site, onde se encontra algumas informações utéis

</footer> → Tag de fechamento do rodapé do site

</html> → Tag de fechcamento para o documento html
